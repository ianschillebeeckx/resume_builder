# CV Add

Add a new entry to the master CV database.

## Instructions

1. Read the current CV database from `data/cv.yaml`
2. Ask what type of entry to add:
   - Experience (job/role)
   - Education
   - Skill
   - Project
   - Certification
3. Gather the relevant details through conversation
4. Add the entry to the appropriate section in the database
5. Add relevant tags for experience/project entries
6. Write the updated database back to `data/cv.yaml`
7. Confirm what was added

## Notes

- For experience entries, help craft strong bullet points (action verb + what + impact)
- Suggest relevant tags based on the content
